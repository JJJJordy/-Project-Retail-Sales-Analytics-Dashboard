# -Project-Retail-Sales-Analytics-Dashboard

The business story: You're a data analyst hired by a retail company. Your job is to analyze their sales data and build an interactive dashboard that helps management answer real business questions like:

- Which products and categories drive the most revenue?
- Which regions are underperforming?
- Are there seasonal sales trends?
- Who are the most valuable customers?

Tech stack: Python (Pandas) → SQLite (SQL queries) → Streamlit (web app)
